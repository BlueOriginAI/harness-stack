# harness-apply

## 用途

按已评审计划执行开发，并持续对账范围、偏差和未完成项。

## 何时使用

- 已完成 `harness-prd` 与 `harness-eng`
- 已有较稳定的实现计划
- 需要进入开发执行阶段
- 需要把结果整理成可交给 `harness-qa` 的摘要

## 输入

- 已评审计划
- 相关代码与上下文

## 输出

- 本轮目标
- 实现范围
- 已完成项
- 关键改动点
- 偏差说明
- 未完成项
- 风险与待验证点
- 交给 `harness-qa` 的输入

## 关联

- `workflows/harness-apply.md`
- `templates/apply/harness-apply.md`
